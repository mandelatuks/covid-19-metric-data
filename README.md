## Getting Started with Create React App

**Installation**
- Clone the repository
~~~
git clone git@github.com:mandelatuks/covid-19-metric-data.git

cd covid-19-metric-data
~~~
- Install NPM packages
~~~
npm install
~~~
- Run server
~~~
npm start
~~~
